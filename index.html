API 端点说明

🤖 AI API

POST /v1/chat/completions
OpenAI兼容的聊天API，支持流式输出
GET /v1/models
获取所有可用的AI模型列表
POST /v1/refresh-models
手动刷新模型列表

📊 监控与健康

GET /monitor
实时监控面板（当前页面）
GET /metrics
Prometheus格式的性能指标
GET /health
基础健康检查
GET /api/health/detailed
详细健康状态，包含0-100分的健康评分

📈 统计与日志

GET /api/stats/summary
24小时内的统计摘要
GET /api/logs/requests?limit=100
获取请求日志（支持分页）
GET /api/logs/errors?limit=50
获取错误日志
GET /api/logs/download?log_type=requests
下载日志文件（requests/errors）

🔍 其他功能

GET /api/request/{request_id}
获取特定请求的详细信息
GET /api/alerts?limit=50
获取系统告警历史
WS /ws
浏览器WebSocket连接端点
WS /ws/monitor
监控面板WebSocket连接

💡 快速使用示例

# Python 示例
from openai import OpenAI
client = OpenAI(
base_url="http://${window.location.hostname}:${window.location.port}/v1",
api_key="sk-any-string-you-like"
)
response = client.chat.completions.create(
model="claude-3-5-sonnet-20241022",
messages=[{"role": "user", "content": "Hello!"}],
stream=True
)
